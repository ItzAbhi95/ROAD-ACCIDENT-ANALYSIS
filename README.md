# Road Accident Dashboard 🚗📊

## Project Overview

This project is an interactive **Road Accident Dashboard** developed to analyze accident and casualty data and identify patterns related to road accidents. The dashboard provides valuable insights into accident severity, vehicle involvement, weather conditions, road surfaces, and geographic accident distribution.

The goal of this project is to transform raw accident data into meaningful visual insights that can help authorities and organizations improve road safety and decision-making.

## Objectives

- Analyze road accident trends and casualty statistics
- Identify factors contributing to fatal accidents
- Compare accident data across different years
- Study accident severity and vehicle involvement
- Create an interactive dashboard for decision-making

---

## Dataset Information

**Source:** Public road accident dataset

The dataset contains information related to:

- Accident severity
- Number of casualties
- Vehicle type
- Weather conditions
- Road surface conditions
- Accident location
- Date and year
- Geographic coordinates
- Road type information

## Key Performance Indicators (KPIs)

## Dashboard Features

### Year Comparison Analysis
- Compare current year and previous year accident trends
- Analyze Year-over-Year (YoY) changes

### Vehicle Type Analysis

- Agricultural Vehicle
- Bus
- Car
- Bicycle

- Van

Accidents analyzed under:

- Fine weather
- Rain
- Snow/Fog

### Road Surface Analysis
Road conditions include:

- Dry roads

- Frost/Snow

### Road Type Analysis
Accident occurrence on:

- Single carriageway
- Dual carriageway
- Roundabout
- Slip road
- One-way streets

### Geographic Analysis
- Interactive map visualization displaying accident locations
- Regional distribution of accidents

## Key Insights

- Most accidents occurred on **dry road surfaces**
- Fine weather conditions had the highest accident percentage
- Single carriageways experienced the highest number of fatal casualties
- Cars were involved in the highest number of casualties among vehicle types
- Slight casualties represented the largest share of total casualties
